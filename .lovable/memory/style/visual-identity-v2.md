- Primary color: Blue (211 100% 50%) — like iMessage

- Light mode: White background, light gray muted

- Rounded pill-shaped tabs like iOS
- Chat list: Large 56px avatars, chevron arrows, separator lines
- Bottom nav: Simple icons, filled when active
- No Pro badges, no crown icons, no subscription UI
- All avatar frames available to everyone
